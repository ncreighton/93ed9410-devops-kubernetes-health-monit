# DevOps Kubernetes Health Monitor Bot

Telegram bot for DevOps teams that monitors Kubernetes cluster health in real time. Reports pod failures, resource exhaustion warnings, and deployment drift directly to Telegram. Supports multi-cluster monitoring, alert routing by severity, and /status /pods /events slash commands.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
